Installation & Inställningar – Frontend

Detta projekt använder Expo för React Native. Följ stegen nedan för att komma igång.

1. Installera beroenden

Installera nödvändiga paket med:

npm install


2. Installera Expo Font (för typsnitt)

npx expo install expo-font


3. Kontrollera projektet

Kör Expo Doctor för att verifiera att allt är korrekt installerat:


4. Starta frontend-projektet

Starta Expo-projektet i utvecklingsläge:

npx expo start -c

-c flaggan rensar cachen (bra om du har problem med laddning).

När servern körs kan du skanna QR-koden i terminalen eller köra appen i en emulator.


Frontend-projektet körs nu via Expo Dev Tools och är redo för utveckling.
